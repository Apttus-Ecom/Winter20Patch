Description

Product configuration component is used to show the hierarchical view of the configurations for a given product or cart line item. This component is responsible for dynamically loading the rich CPQ UI experience or the native client experience based on the configuration layout defined on the storefront.

Usage

import { ProductConfigurationModule } from '@apttus/elements';

@NgModule({
  imports: [ProductConfigurationModule, ...]
})
export class AppModule {}

Implements

OnInit

Example

<apt-product-configuration
   [productId]="product.Id"
   [relatedTo]="parent"
></apt-product-configuration>
